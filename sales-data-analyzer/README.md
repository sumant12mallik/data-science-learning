# Sales Data Analyzer (Python)

This project analyzes sales data using core Python concepts like:
- loops
- conditions
- functions
- data categorization
- insight generation

## Features
- Ignores negative values (invalid data)
- Counts zero sales days
- Calculates total, average, min, max
- Categorizes sales into Low / Mid / High
- Generates business insights automatically
- Works with multiple datasets (branches)

## Example Output
- Total sales
- Average sales
- Best and worst performance category
- Actionable insights for decision making

## Concepts Used
- for loop
- if-elif-else
- functions
- dictionaries
- clean coding practices

## How to Run
```bash
python analyze_sales.py
